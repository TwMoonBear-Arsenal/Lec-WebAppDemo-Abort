# Lec-WebAppDemo

## 說明資訊
### 模組目的

* 提供WebApp教學使用，說明下列概念：
  * 乾淨架構：Clean WebApp
  * 控制反轉：Inversion of Control
  * 依賴注入：Dependency Inkection  
* 依據Clean WebApp架構，分別提供下列層次功能：
  * L5.操作介面：網頁介面
  * L4.操作服務：網頁介面功能
  * L3.核心類別：核心類別
  * L2.基建服務：資料庫連接服務
  * L1.基建項目
